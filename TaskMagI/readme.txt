                                         VisierImporter beta
Программа для импорта данных из бланка задания по средствам считывания QR-кода, в базу данных МАГ.

Для работы программы потребуется установка Java 6 или старше

Настройки БД хранятся в файле config.xml

Алгоритм работы с программой:
1. Считывание QR кода или копирывание его в верхнее левое поле.
2. Нажимаем кнопку "Разобрать!"
3. Сверяем, если необходимо корректируем данные, в полях находящихся в правой части оконного интерфейса.
4. Нажимаем кнопку "Импортировать!"
5. Profit! (ловим ошибки)
